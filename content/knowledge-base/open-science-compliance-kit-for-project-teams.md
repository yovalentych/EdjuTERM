## Мінімальний комплект

- дата-менеджмент план;
- словник даних;
- версіонований протокол;
- README для набору даних;
- ліцензія;
- правила доступу;
- історія змін і аудиту.

## Що має бути готово до публікації набору

| Артефакт | Навіщо потрібен |
|---|---|
| README | дає контекст і спосіб використання |
| Data dictionary | пояснює поля та кодування |
| License | визначає умови повторного використання |
| Provenance | показує джерело і трансформації |
| Methods note | пояснює як отримані результати |

## Практичний порядок дій

1. Визначити, що є відкритим, що під ембарго, що обмежене.
2. Підготувати FAIR-метадані раніше, ніж сам release.
3. Перевірити конфіденційність і права на дані.
4. Зібрати пакет файлів для репозиторію.
5. Додати короткий public summary людською мовою.
