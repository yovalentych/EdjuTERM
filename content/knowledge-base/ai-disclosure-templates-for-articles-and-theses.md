## Коли який шаблон використовувати

| Випадок | Найкраще місце |
|---|---|
| ШІ був частиною дослідницького процесу | Methods |
| ШІ допомагав у підготовці рукопису | Acknowledgements |
| Кафедра або журнал вимагає окремий блок | AI declaration |
| Потрібно явно зазначити, що ШІ не використовувався | Декларація use / no use |

## Шаблон для Methods

> У цьому дослідженні використовувався інструмент [назва, версія, розробник] у період [дати] для [конкретне завдання]. Виходи інструмента були незалежно перевірені авторами шляхом [опис перевірки]. Інструмент не використовувався для фабрикації даних, генерування неперевірених джерел або автоматичного формування фінальних висновків.

## Шаблон для Acknowledgements

> Під час підготовки рукопису автор(и) використав(ли) [назва інструмента, версія, розробник] для [мовне редагування / покращення читабельності / первинне структурування тексту / переклад]. Після цього автор(и) самостійно переглянув(ли), відредагував(ли) та верифікував(ли) текст і несе(уть) повну відповідальність за зміст публікації.

## Шаблон для окремої AI declaration

> У роботі використано [назва / версія / розробник] для [мета]. ШІ не використовувався для отримання емпіричних результатів без авторської перевірки, формування наукових висновків без критичного аналізу або створення неперевірених бібліографічних посилань. Усі фрагменти, для яких застосовувався ШІ, були перевірені та відредаговані автором(ами).

## Шаблон для декларації про невикористання

> Під час підготовки цієї роботи інструменти генеративного штучного інтелекту не використовувалися.

## Як писати коректно

- не використовувати розмиті формули без конкретики;
- завжди називати інструмент і його роль;
- прямо вказувати, що перевіряв автор;
- не змішувати технічне мовне редагування з дослідницьким застосуванням ШІ.
